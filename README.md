# Café Solo - Montpellier (Version Multi-pages)

Site internet premium et responsive pour le coffee shop **Café Solo** à Montpellier.

## 📁 Structure des fichiers
- `index.html` : Page d'accueil (Visuels majeurs, accroches et Storytelling/Histoire).
- `ateliers.html` : Section Événements, Ateliers Latte Art et formulaires de réservation.
- `boutique.html` : Vitrine des produits (cafés en grain, accessoires, céramiques).
- `services.html` : Présentation des services (Wi-Fi, Coworking, Brunch).
- `galerie.html` : Grille d'ambiance visuelle du coffee shop.
- `contact.html` : Informations pratiques, horaires et formulaire de contact.
- `cookies.html` : Page légale sur la politique de gestion des cookies.
- `dashboard.html` : Espace d'administration pour la gestion de l'activité.
- `style.css` : Feuille de style globale et unique (gère le responsive et le Dark Mode).
- `script.js` : Logique globale (Menu mobile hamburger, ouverture de la carte, sélecteur de thème).

## 🚀 Utilisation
Copiez tous les fichiers dans un **même dossier** et lancez `index.html` dans votre navigateur.
